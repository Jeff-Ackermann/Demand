# Demand App

This folder contains the Demand practice app in two versions:

- `index.html`: desktop chalkboard version
- `index_mobile.html`: phone-friendly version

## Recommended student link

For students using phones, share the direct GitHub Pages URL for:

- `index_mobile.html`

Example:

`https://YOUR-USERNAME.github.io/demand/index_mobile.html`

## Publish on GitHub Pages

1. Create a new GitHub repository, such as `demand`.
2. Upload the files from this folder to the repository root.
3. On GitHub, open `Settings`.
4. Open `Pages`.
5. Under `Build and deployment`, set:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/ (root)`
6. Save.
7. Wait for GitHub Pages to publish the site.

Your site will usually appear at:

- `https://YOUR-USERNAME.github.io/REPOSITORY-NAME/`

For this app, the most useful links are:

- Desktop app: `https://YOUR-USERNAME.github.io/REPOSITORY-NAME/index.html`
- Mobile app: `https://YOUR-USERNAME.github.io/REPOSITORY-NAME/index_mobile.html`

## Suggested sharing workflow

- Put the mobile URL in D2L/Canvas
- Put the mobile URL into a QR code for lecture slides
- Keep the desktop URL for classroom demos
